📥 نصب و راه‌اندازی (فقط بار اول):
================================================================================

۱. Python رو نصب کن (اگه نداری)
   - برو به سایت python.org
   - دکمه زرد Download Python رو بزن
   - فایل دانلود شده رو باز کن
   - ⚠️ حتماً تیک "Add Python to PATH" رو بزن
   - Install Now رو بزن

۲. کتابخونه‌ها رو نصب کن
   - روی فایل install.bat دابل کلیک کن
   - خودکار همه چی نصب میشه
   - صبر کن تا پیغام "Installation Complete" رو ببینی
   - یه دکمه بزن تا بسته بشه

۳. برنامه رو اجرا کن
   - روی فایل run.bat دابل کلیک کن
   - برنامه باز میشه!

   ⚠️ توجه: دفعات بعد فقط run.bat رو اجرا کن
   ⚠️ دیگه نیازی به install.bat نیست


⚙️ تنظیمات اولیه (فقط بار اول):
================================================================================

۱. ⚙️ دکمه تنظیمات (چرخ‌دنده) رو بزن

۲. 🎯 قسمت Game Window:
   - اول بازی رو باز کن
   - از لیست کشویی، اسم پنجره بازیت رو پیدا کن و انتخاب کن همییشه اول پنجریی که میخوای توش خودکار بنویسه از اینجا انتخواب کن
   - اگه بازی تو لیست نیست، دکمه ↻ (Refresh) رو بزن

۳. ⏱️ سرعت تایپ رو تنظیم کن (با دکمه + و -)
   - 0.03 = سریع
   - 0.08 = طبیعی
   - 0.15 = کند

۴.  "Auto-send with Enter"   اگه دوست نداری خودش خودکار پیام بفرسته و خودت هر وقت خواستی ارسالش کنی تیک این گذینه رو بردار اینجوری فقط برات ترجمه میکنه و اگه صلاح دونستی خودت ارسال میزنی که پیام بره

۵. 💾 دکمه Save Settings  یادت باشه همیشه تو تنظیمات هر کاری کردی باید رو بزن


🎯 طرز استفاده (هر دفعه):
================================================================================

⚠️ به ترتیب انجام بده:

۱. اول بازی رو باز کن
۲. برو تو محیط بازی (جایی که چت هست)
۳. روی run.bat دابل کلیک کن (یا اگه برنامه بازه، Ctrl+/ رو بزن)
۴. شخصیت رو انتخاب کن:
   👤 Normal  = ترجمه معمولی
   😎 Gamer   = اسلنگ گیمری (yo, pog, sheesh)
   😈 Toxic   = ترش‌تاک (ez, skill issue)
   🗿 Sigma   = خونسرد و مرموز
   🥶 Chill   = بی‌خیال و ریلکس
۵. زبان مقصد رو انتخاب کن (🇬🇧 English یا هر چی دوست داری)
۶. متن فارسی رو تو باکس بنویس
   مثال: سلام خوبی؟ بیا بریم
۷. ⚠️ با موس برو رو قسمتی که چت هست باید روی جاییه قرار بنویسی چت باز باشه یا یه بار کلیک کن روی قسمت جایی که قراره بنویسی  بازی کلیک کن (مهم!)
۸. Ctrl+1 رو بزن
۹. تمام! برنامه خودکار ترجمه میکنه، تایپ میکنه،


⌨️ کلیدهای میانبر:
================================================================================

   Ctrl+/     = نشون دادن/مخفی کردن مترجم
   Ctrl+1     = ترجمه و ارسال

⚠️ مشکلات رایج:
================================================================================

❌ برنامه اجرا نمیشه
✅ Python رو با تیک Add to PATH نصب کردی؟
   install.bat رو یه بار اجرا کن

❌ ترجمه نمیکنه
✅ اینترنت وصله؟
  

❌ تو بازی تایپ نمیشه
✅ تنظیمات > پنجره بازی رو انتخاب کن
   قبل از Ctrl+1 با موس رو بازی کلیک کن

❌ پنجره مترجم نیست
✅ Ctrl+/ رو بزن
   Alt+Tab بزن بگرد


💡 نکات مهم:
================================================================================

• همیشه قبل از Ctrl+1 با موس رو جایی که قرار بنویسی  کلیک کن
• جمله رو ساده بنویس
• با موس می‌تونی پنجره رو بکشی هر جا
• تنظیمات فقط بار اول، دفعات بعد خودکار هست
• دکمه ✕ مخفی میکنه، Ctrl+/ برمیگردونه
